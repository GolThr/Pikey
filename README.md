# Pikey - Piano Keyboard

使用连接到安卓设备的键盘演奏音乐
可自定义SoundFonts音源

![image](https://github.com/GolThr/Pikey/blob/main/assets/image.jpg)

### 已有功能
1. 键盘按键映射
2. 触摸琴键
3. 键盘，琴键同步
4. 切换键盘布局
5. 增加键盘布局
6. 调性调整
7. 延音控制
8. 自定义键盘控件
9. 实时可视化按键
